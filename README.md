# What is a Fibonacci Series?

The Fibonacci Series is a sequence of numbers in which each number is the sum of the two previous numbers.

# How the Program Works

1. n = 10 specifies that the program will generate 10 Fibonacci numbers.
2. first = 0 and second = 1 are initialized as the first two numbers of the Fibonacci sequence.
3. Inside the for loop:
   * third = first + second calculates the next Fibonacci number.
   * The value of third is printed.
   * first is updated to second.
   * second is updated to third.
4. This process repeats until 10 Fibonacci numbers are generated.
